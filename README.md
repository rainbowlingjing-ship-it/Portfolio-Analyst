# PD Model Development, Validation and Backtesting

## Project Overview

This project develops a Probability of Default (PD) model using logistic regression to estimate borrower default risk.

The model includes:

* Data preprocessing and feature engineering
* Portfolio segmentation for risk differentiation
* Model validation using AUC and KS statistic
* Out-of-time (OOT) validation
* Backtesting and calibration analysis
* Population Stability Index (PSI) monitoring

## Dataset

Simulated credit risk dataset with borrower financial and behavioral characteristics.

## Tools

* Python (pandas, numpy, scikit-learn)
* Logistic Regression
* Risk modeling techniques (AUC, KS, PSI)

## Key Results

* Strong risk ranking across segments
* Stable performance between development and OOT samples
* Low PSI indicating minimal population shift
